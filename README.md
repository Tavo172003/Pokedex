# Pokedex
Pokedex interactiva con estética retro pixel-art hecha en HTML, CSS y Vanilla JS. Consume la PokeAPI optimizando la carga por bloques. Permite buscar, filtrar por tipo/generación y abrir un modal asíncrono con estadísticas, cadena evolutiva detallada, ubicaciones de captura y reproducción del grito original del Pokémon.
